# 🚗 Car Rental Landing Page

A modern car rental landing page designed using **HTML & CSS**, focusing on clean, responsive, and user-friendly design.

## 📌 Project Overview
This project is a landing page for a car rental service, including the following sections:
- Home
- Services
- Car Models
- Blog
- Footer with links and newsletter subscription

## 🛠️ Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design
- Media Queries

## ✨ Project Features
- Modern and clean UI design  
- Fully responsive layout  
- Hover effects and smooth transitions  
- Well-structured and organized code
